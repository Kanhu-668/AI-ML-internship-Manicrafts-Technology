# California House Price Prediction 🏠

## Project Overview
This project builds a Machine Learning model to predict California housing prices using the famous California Housing dataset.

The goal is to understand the end-to-end ML pipeline:
- Data preprocessing
- Feature selection
- Model training
- Model evaluation

## Dataset Features
- MedInc (Median Income)
- HouseAge
- AveRooms
- AveBedrms
- Population
- AveOccup
- Latitude
- Longitude

Target variable:
- Median House Value

## ML Workflow
1. Data Cleaning & Exploration
2. Feature Selection
3. Train/Test Split
4. Model Training
5. Model Evaluation using R² score

## Models Used
- Linear Regression

## Result
Model R² Score: **0.50+**

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Scikit-learn, Jupyter Notebook
